# 豆听（DouListen）

豆听 是一个浏览器扩展，当你浏览豆瓣的图书、电影或创作者页面时，
自动为你展示与当前内容相关的播客单集，帮助你从“阅读 / 观影”自然延伸到“收听”。



## 🔗 下载地址
edge：https://microsoftedge.microsoft.com/addons/detail/mppamoanbcdflmkkkaaimkgkjmfbebap

## ✨ 功能介绍

- 📚 豆瓣图书页：展示讨论该书的播客单集
- 🎬 豆瓣电影页：推荐影评 / 访谈相关播客
- 👤 创作者页面（作家 / 导演 / 演员）：聚合相关播客内容
- 🔍 支持关键词搜索与分页浏览
- 🎧 不采集、不存储任何用户数据



## 🔒 隐私说明

- 本扩展不会收集、存储或上传任何用户个人信息
- 所有数据仅在当前页面内即时使用
- 不包含任何跟踪、统计或广告行为
